#####
About
#####

|Eco|_ is a software package for Bayesian inference of shared evolutionary
events.
|Pyco|_ is a companion Python package for summarizing the output of |eco|_.
|Eco|_ and |pyco|_ are developed by the |phyleticalab|_.
